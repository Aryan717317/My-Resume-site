Selected Works
No. I Flow State AI Pipeline
A real-time focus-monitoring pipeline — Chrome extension, through a Flask event server, into an activity processor, concluding with an LLM reasoner. The provider-agnostic client supports both local inference via Ollama and cloud inference through AWS Bedrock or Claude, with session state persisted across restarts.
Materials — Python · Flask · Claude API · AWS Bedrock · Ollama
No. II Doceader RAG Pipeline
An end-to-end document inquiry pipeline, proceeding from ingestion through chunking, embedding, retrieval, and generation. Constructed upon LangChain, ChromaDB, and FastAPI, with particular attention paid to retrieval relevance and the latency each stage imparts upon final output.
Materials — LangChain · ChromaDB · FastAPI · HuggingFace
No. III Synoptic AI Multi-Agent
A multi-agent orchestration platform, raised upon Google ADK, Gemini, and FastAPI. Coordinates several specialised agents working concurrently upon a shared task, governed by a dedicated routing and inter-agent communication layer.
Materials — Google ADK · Gemini API · FastAPI · Python
No. IV GestureShare Peer-to-Peer
A peer-to-peer file-sharing utility, governed by hand-gesture control. Accompanied by a nine-document technical specification — state machines, wire protocol, platform guides, and testing strategy — composed in full before implementation began.
Materials — Go · Tauri · SvelteKit · P2P Protocol
