# Aryan Bharat Kumar - Portfolio Website

An interactive, old-paper-styled portfolio website built for showcase and play.

## Features
- **Profile**: A quick introduction, provenance/background, and core philosophy.
- **Works**: Catalogue of selected projects, mini-games, and a wheel of fortune.
- **Ledger**: Inventory of languages, tools, frameworks, and certifications.
- **Journal**: Timeline of recent work and current studies.
- **Contact**: Correspondence form for reaching out.
- **Secrets & Easter Eggs**: Hidden interactive features, including Konami Code support.

## Technologies Used
- HTML5 (Standardized Semantic Markup)
- Vanilla CSS3 (Custom Typography, Animations, Responsive Layouts)
- Modern JavaScript (Interactive elements, game logic, easter egg handling)
